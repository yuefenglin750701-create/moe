實驗室特色與目標

負責系所：智慧自動化工程系

本實驗室致力於軟硬體虛實整合，特別強調 Python 程式設計在工業現場的應用。 我們專注於將 CNC 加工參數進行數據化分析，透過 IoT 物聯網技術實現機台數據的即時擷取與可視化。 這裡是跨域人才的培育基地，讓機械背景學生學習程式邏輯，讓資工背景學生接觸實體加工。

  • 數據可視化：開發機台戰情室看板，即時監控生產狀態。
  • Python 訊號處理：分析主軸負載與振動訊號，建立異常預警模型。
  • 跨域整合專題：結合感測器與控制器，進行加工參數優化研究。
