多軸加工實驗室 (Multi-Axis Machining Lab)

多軸加工實驗室全景

五軸同動綜合加工機與控制器

場域特色

本實驗室整合了傳統加工設備與先進五軸同動機，搭配 CNC 程式模擬器、3D CAD/CAM 軟體與 NC 模擬平台。讓學生能在實際切削前先進行軟體模擬與路徑驗證，再接軌實體操作，達到「虛實整合」的教學效果。

主要設備：

  • 五軸同動綜合加工機
  • CNC 綜合加工切削中心
  • 智慧型 CNC 程式模擬器 (5台)
  • Mastercam 3D CAD/CAM 軟體
  • 多軸控制資料記錄系統

支援課程

  • 五軸加工技術
  • 工具機導論
  • 虛實整合技術應用
每週使用時數 9 小時

智慧多軸複合技術實驗室 (Intelligent Multi-Axis Tech Lab)

IoT 感測器與 Python 開發平台

數據分析與視覺化教學環境

場域特色

強調 Python 程式設計與數據視覺化應用，提供適合零基礎學生的入門實作環境。實驗室支援加工參數分析、IoT 資料整合與 AI 模型訓練，結合通識與專業課程，推動全校性的跨域程式教育。

主要設備：

  • Python 開發平台 (PC端 20台)
  • 加工模擬控制平台 (模擬進給、轉速數據)
  • 工業感測模擬器 (支援 JSON/CSV 輸出)
  • Jupyter Notebook 教學環境

支援課程

  • Python 程式設計
  • 訊號處理與影像辨識
  • 感測器聯網與數據分析
每週使用時數 6 小時
